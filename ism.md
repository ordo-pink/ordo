# ISM

**International Simplified Markup**

## Brief

### Block Elements

- paragraph - text
- line break - an empty line
- headers
  - +++++ text - <h1>text</h1>
  - ++++ text - <h2>text</h2>
  - +++ text - <h3>text</h3>
  - ++ text - <h4>text</h4>
  - - text - <h5>text</h5>
- details
  - text // hidden text <details><summary>text</summary><div>hidden text</div></details>
